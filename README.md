# Science Archive Client

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints files and prints out lint errors
```
npm run lint:check
```

### Lints and fixes files
```
npm run lint:fix
```
